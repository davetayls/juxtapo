Global community

Juxtapo is still in it's forming stage and is looking for experienced developers to build on it's solid base. This is your chance to make a real impact on the direction of an open source project and we need you to help us move it forward.

We're especially looking for: JavaScript developers, testers, visionaries.
To get started go to the project on github and create your own fork.

Juxtapo community

Juxtapo has been designed with a small basic core and to be extremely easy to extend. This is so that we put YOU the developer in control.

But we don't want you to keep it to yourself, if you have developed a plugin we would like to share it on this site. Please email us your plugins.
