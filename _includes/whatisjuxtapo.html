What is juxtapo?

You can test Juxtapo on this site using the controller at the bottom left of the screen.

1 In-browser tools using JavaScript to assist while you're turning designs in to html/css

The core library is designed to be basic and fast to get up and running.

  • Overlays
  • Keyboard nudging
  • Page auto refresh

2 A simple plugin API which allows for any developer to extend the core functionality.
